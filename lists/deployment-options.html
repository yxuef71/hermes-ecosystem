breakdown

numtide / llm-agents.nix

This project provides a comprehensive collection of Nix packages for various AI coding agents, including Hermes. It features automated daily updates and supports both open-source and proprietary binary-distributed agentic tools.

rookiemann / portable-hermes-agent

Designed for Windows users, this zero-install desktop agent includes a GUI, 100 built-in tools, and support for local models via LM Studio. It functions without Docker, offering a dynamic tool maker and workflow engine for custom automation.

TheAiSingularity / hermesclaw

This solution focuses on high-security environments by sandboxing Hermes Agent with hardware-level enforcement via NVIDIA OpenShell. It offers three security policy presets and a pre-built skills library for tasks like research digests and code reviews.

xmbshwll / hermes-agent-docker

This project offers a streamlined Docker sandbox image that includes the official Hermes installation and mini-swe-agent. It supports persistent state through mounted home directories and allows for specific versioning via build arguments.

0xrsydn / nix-hermes-agent

This NixOS module enables declarative and reproducible deployments of Hermes, managing configurations, documents, and systemd services. It ensures secure secrets handling through sops-nix integration and supports hybrid skill management.

JackTheGit / hermes-autonomous-server

Targeted at production environments, this project provides a headless systemd deployment for Ubuntu and Debian servers. It configures Hermes Gateway as a reboot-safe service to enable the autonomous execution of recurring AI cron jobs.

jpalmae / hermeshq

HermesHQ acts as a Docker-first control plane for orchestrating multiple Hermes Agent instances through a single web application. It enables hierarchy-aware inter-agent delegation, task scheduling, and enterprise MCP access.

ellickjohnson / portainer-stack-hermes

This deployment method uses a Docker Compose stack integrated directly with Portainer as a Git Repository stack. It features a ttyd web terminal interface and automates image builds using GitHub Actions.
