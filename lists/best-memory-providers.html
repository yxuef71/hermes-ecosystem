breakdown

mem0ai / mem0

As the official memory provider for Hermes Agent, Mem0 offers a universal memory layer that manages states across users, sessions, and autonomous agents. It utilizes hybrid retrieval and temporal reasoning to rank past and future events, ensuring agents maintain a time-aware understanding of their interactions.

garrytan / gbrain

This opinionated 'agent brain' synthesizes cited answers by automatically extracting typed entity relationships into a self-wiring knowledge graph. It is designed for multi-tenant deployments, allowing for scoped data access across people, companies, and meeting notes.

vectorize-io / hindsight

Hindsight implements a long-term learning framework through dedicated retain, recall, and reflect workflows. It provides an enterprise-ready memory solution with a two-line LLM wrapper and support for PostgreSQL and Oracle AI Database storage.

plastic-labs / honcho

Honcho serves as a sophisticated library that extracts reasoning-based conclusions from conversations rather than relying on simple vector matching. It tracks the evolution of entities like users and projects using a combination of BM25 and vector search for deep session context.

greyhaven-ai / autocontext

This recursive context harness uses a five-role architecture to help agents succeed on complex tasks through iterative self-improvement. It focuses on knowledge persistence and structured evaluation, producing reusable playbooks and detailed execution traces.

stephenschoettler / hermes-lcm

Hermes-LCM is a lossless context management plugin that uses a hierarchical DAG structure to prevent message loss during context compaction. It features a SQLite-backed store and specialized agent tools for targeted retrieval and summary expansion.

AxDSan / mnemosyne

Mnemosyne is a zero-dependency memory system built on SQLite that delivers sub-millisecond latency for reads and writes. Its MEMORIA engine performs structured fact extraction and recursive gap analysis to ensure memory consistency.

elkimek / honcho-self-hosted

This project provides a self-hosted backend for the Honcho library, enabling private cross-session persistence for Hermes Agent. It includes automated setup scripts for Docker and Redis, supporting any OpenAI-compatible or local inference provider.

yoloshii / ClawMem

ClawMem is a local-first memory engine that eliminates cloud dependencies and API keys for privacy-conscious agent deployments. It integrates via MCP and Claude Code, utilizing a hybrid of BM25, vector search, and knowledge graphs.

plur-ai / plur

Plur introduces a shared memory layer using an open YAML 'engram' format for zero-cost local storage in multi-agent systems. It employs ACT-R activation logic to strengthen relevant facts while decaying stale information over time.

EfficientContext / ContextPilot

ContextPilot optimizes long-context inference by reordering and deduplicating shared context blocks to reduce input tokens. It significantly boosts prefill throughput for Hermes Agent and OpenClaw, offering 1.5–3× faster processing speeds.

keepnotes-ai / keep

Keep provides reflective memory by combining BM25 full-text search with ranked graph traversal for deep semantic retrieval. It automatically indexes local files and URLs, using edge tags to create navigable relationships with inverse links.

yepyhun / Brainstack

Brainstack consolidates temporal continuity, graph truth, and corpus retrieval into a single kernel for Hermes Agent. It utilizes a unique five-shelf architecture to organize data and prevent the 'flat memory blob' issues common in simpler systems.

amanning3390 / flowstate-qmd

This anticipatory memory system prefetches context by monitoring agent session logs to predict future needs. It is a local-first, MCP-native server that uses SQLite-vec and GGUF models for efficient on-device retrieval.

alias8818 / hermes-tool-slimmer

This utility reduces token overhead by using BM25 ranking to select only the most relevant tool schemas for a given task. It includes a real-time dashboard to track token savings across native Hermes tools and MCP integrations.
