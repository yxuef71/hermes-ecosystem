breakdown

garrytan / gbrain

This project implements a self-wiring knowledge graph that combines vector retrieval with automated entity extraction for highly structured memory. It is a top-tier choice for power users, exposing over 30 MCP tools to integrate structured memory directly into Claude Code and Cursor workflows.

vectorize-io / hindsight

Hindsight focuses on long-term agent learning through biomimetic data structures that facilitate retain, recall, and reflect workflows. It allows Hermes agents to build evolving mental models and facts over time, supporting integration across major providers like OpenAI and Ollama.

greyhaven-ai / autocontext

This recursive context harness uses a multi-agent loop—including roles like coach and architect—to accumulate validated knowledge across repeated runs. It is designed to help agents succeed on complex investigations by utilizing verifier-driven checkpoints and browser-integrated research.

stephenschoettler / hermes-lcm

Hermes-LCM provides a lossless context engine that uses hierarchical DAG summaries and immutable SQLite storage to ensure no message data is ever deleted. It offers a three-level escalation system for context compaction, allowing agents to grep and expand history without losing critical details.

elkimek / honcho-self-hosted

This self-hosted backend enables private, cross-session memory for Hermes agents through a robust stack of Docker, PostgreSQL, and Redis. It is the ideal solution for developers requiring full control over their memory infrastructure while maintaining compatibility with any OpenAI-style LLM provider.

yoloshii / ClawMem

ClawMem is a local-first context engine that uses a unified SQLite vault to persist memory across different agents without cloud dependencies. It features a hybrid retrieval system combining BM25, vector search, and graph traversal to automate decision capture and handoffs.

plur-ai / plur

Plur introduces a shared memory layer using an open YAML 'engram' format, making agent memories human-readable and portable. It leverages zero-cost local retrieval through hybrid embedding search to automate memory injection for Hermes and other MCP-compatible tools.

AxDSan / mnemosyne

Mnemosyne is a zero-dependency memory system optimized for sub-millisecond retrieval using a native SQLite-vec and FTS5 hybrid search. Its unique three-tier BEAM architecture separates working, episodic, and scratchpad memory for high-performance local agent operations.

keepnotes-ai / keep

Keep provides reflective memory by indexing codebases and media with automatic OCR and summarization. It organizes information using bidirectional edge tags, creating a navigable web of relationships accessible via MCP and CLI interfaces.

amanning3390 / flowstate-qmd

This anticipatory memory system prefetches context by monitoring agent session logs to predict and cache relevant information. Built as an MCP-native server, it utilizes local GGUF models and SQLite-vec to provide a low-latency, 'future-aware' RAG experience.
