breakdown

garrytan / gbrain

This opinionated brain uses a self-wiring knowledge graph to automatically extract entities and relationships from raw data. It features a hybrid search system that combines vector RAG with graph-based ranking, supported by 28 specialized skills for autonomous data enrichment.

vectorize-io / hindsight

Hindsight implements biomimetic data structures to facilitate long-term learning through dedicated retain, recall, and reflect workflows. It allows Hermes to build evolving mental models and integrates seamlessly with major providers like OpenAI, Anthropic, and Ollama.

greyhaven-ai / autocontext

Designed for complex tasks, this recursive harness uses a multi-agent loop—including roles like coach and architect—to accumulate validated knowledge over repeated runs. It ensures mission success through verifier-driven checkpoints and browser-integrated investigations.

stephenschoettler / hermes-lcm

This plugin provides lossless context management by storing an immutable history in a hierarchical DAG structure within SQLite. It uses a three-level escalation system for context compaction, allowing agents to grep or expand history without ever deleting original message data.

elkimek / honcho-self-hosted

This project offers a private, self-hosted backend for Honcho memory, enabling cross-session persistence for Hermes agents. It includes an automated setup script for Docker, PostgreSQL, and Redis, supporting any OpenAI-compatible or local inference server.

yoloshii / ClawMem

ClawMem is a local-first memory engine that uses a unified SQLite vault to persist context across different agents without cloud dependencies. It utilizes a hybrid retrieval approach—combining vector, BM25, and graph traversal—to automate decision capture and handoffs.

plur-ai / plur

Plur introduces a shared memory layer using an open YAML 'engram' format for easy local storage and portability. It provides zero-cost local retrieval through hybrid BM25 and embedding search, automating memory injection for Hermes and MCP-compatible tools.

AxDSan / mnemosyne

Focusing on extreme performance, Mnemosyne delivers sub-millisecond latency for memory operations using a zero-dependency SQLite architecture. Its BEAM architecture organizes data into tiered working, episodic, and scratchpad memory segments based on importance scores.

amanning3390 / flowstate-qmd

This anticipatory memory system prefetches relevant context by monitoring agent session logs in real-time. Built as an MCP-native server, it utilizes SQLite-vec and local GGUF models to provide low-latency RAG capabilities for coding agent clients.
