breakdown

junhoyeo / tokscale

This Rust-based TUI tracks token consumption and real-time costs across over 20 AI coding agents, including Hermes and Claude Code. It utilizes LiteLLM pricing data and 3D visualizations to provide developers with granular oversight of their model expenditure.

joeynyc / hermes-skins

This utility allows users to personalize the Hermes terminal interface through YAML-based themes that modify colors, ASCII art, and spinner animations. It supports partial skin inheritance and includes a library of presets ranging from retro DOS aesthetics to sci-fi themes like Skynet.

AlexAI-MCP / hermes-CCC

This project ports the Hermes Agent brain and 46 core skills directly into the Claude Code Channel, enabling native execution without external processes or OAuth. It specifically enhances the environment with MLOps capabilities for vLLM inference, GGUF management, and reinforcement learning training.

unmodeled-tyler / vessel-browser

Vessel is an AI-native browser designed for autonomous agent control, exposing structured page schemas and element visibility through the Model Context Protocol (MCP). It features durable session checkpoints and a supervisory sidebar, allowing developers to audit and intervene in automated web workflows.

Abruptive / Ankh.md

Ankh.md is a framework for deploying scoped Hermes Agents that are restricted to specific project folders with local configurations and memories. It enables seamless switching between global and project-local agents, ensuring that tools and instructions remain relevant to the immediate workspace.

0xNyk / openclaw-to-hermes

This migration utility streamlines the transition from OpenClaw to Hermes Agent by automatically mapping personas, memories, and workspace files. It handles the complex conversion of model configurations and credentials into the Hermes YAML format to preserve existing agent setups.

roli-lpci / lintlang

This static linter optimizes agent performance by identifying structural language bugs in system prompts and tool descriptions using seven specialized detectors. It operates with zero latency and no LLM calls, providing an automated quality gate for Hermes configurations within CI/CD pipelines.

42-evey / evey-setup

This deployment script automates the setup of a full Hermes Agent stack using Docker, LiteLLM, and Qdrant for local GPU inference. It comes pre-configured with 29 community plugins for memory and social integration, facilitating a zero-cost entry point using free model routing.
