breakdown

junhoyeo / tokscale

This high-performance Rust TUI tracks token consumption and costs across multiple AI coding environments, including Hermes Agent and Claude Code. It provides local data visualization to help developers monitor their spend and usage patterns across various model providers.

joeynyc / hermes-skins

This utility allows developers to personalize the Hermes terminal interface through YAML-based configuration files for banners, spinners, and ASCII art. Themes can be easily shared and created by inheriting from default settings to support partial skin definitions.

AlexAI-MCP / hermes-CCC

This project ports the Hermes Agent brain and 46 core skills directly into the Claude Code Channel for native execution without external processes or OAuth. It specifically includes MLOps tools for vLLM, GGUF inference, and reinforcement learning training.

unmodeled-tyler / vessel-browser

Vessel is an AI-native browser designed for autonomous control via MCP, allowing Hermes Agent to navigate the web with human-in-the-loop approvals. It features automated page schema inference to help agents detect structured entities and forms during live sessions.

Abruptive / Ankh.md

Ankh.md is a framework for creating scoped Hermes Agents that are restricted to specific folders with their own local configurations and tools. It enables seamless switching between global and project-local agents, each with unique memories and custom instructions.

roli-lpci / lintlang

This static linter uses regex and AST analysis to validate agent configurations and system prompts without incurring LLM costs. It identifies seven specific failure modes, such as ambiguity and missing constraints, to ensure prompts meet HERM v1.1 scoring standards.

42-evey / evey-setup

This deployment stack uses Docker to bundle Hermes Agent with LiteLLM, Ollama, and Qdrant for a zero-cost local setup. It comes pre-configured with 29 community plugins and automated routing for free models and local GPU inference.

0xNyk / openclaw-to-hermes

This migration utility streamlines the transition from OpenClaw to Hermes Agent by mapping personas, memories, and skills into the Hermes YAML format. It automates the transfer of complex model configurations and credentials from local environment files.
