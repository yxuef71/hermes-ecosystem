breakdown

nesquena / hermes-webui

This zero-build web interface provides full feature parity with the Hermes Agent CLI through a streamlined three-panel layout for chat, file browsing, and session management. It offers a lightweight, mobile-friendly way to interact with the agent using only Python and vanilla JavaScript.

outsourc-e / hermes-workspace

This security-hardened workspace emphasizes agent orchestration through a dedicated memory browser and skill manager. It utilizes real-time SSE streaming for chat and tool execution, incorporating mandatory approval prompts and authentication middleware for safer operations.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard focused on multi-channel orchestration, allowing users to configure and manage Hermes across Telegram, Discord, Slack, and WhatsApp. It features advanced real-time analytics for token tracking and cost estimation alongside an integrated web terminal for backend management.

dodo-reach / hermes-desktop

A native macOS application that establishes direct SSH connections to Hermes without requiring a middle-layer API or remote daemon. It features a multi-tabbed terminal and specialized tools for managing remote SQLite session stores and environment variables.

xaspx / hermes-control-interface

Acting as a multi-agent gateway, this self-hosted dashboard allows users to start, stop, and configure multiple agent profiles from a single interface. It provides detailed token analytics and a browser-based terminal to monitor system metrics and cron jobs.

fathah / hermes-desktop

A desktop companion app that simplifies the initial setup of Hermes Agent through guided installation and dependency resolution. It serves as a configuration hub for connecting various LLM providers and messaging gateways like Discord and Telegram.

clawvader-tech / hermes-telegram-miniapp

Designed as a mobile-first React SPA for the Telegram Mini App ecosystem, this interface provides a terminal-style chat with slash command support. It allows users to spawn background agents and monitor live system health gauges for CPU and memory usage directly from their phone.

Web3CZ / Web3Hermes

This localized Chinese interface offers a tailored experience for regional users with full support for dark and light themes. It includes a robust workspace file browser for direct editing and a comprehensive session management system with tool call visualization.

sanchomuzax / hermes-webui

This dashboard focuses on process monitoring and historical inspection, featuring FTS5 search capabilities for deep conversation history analysis. It provides a read-only view of configurations and tracks cost distribution across active sessions.

Euraika-Labs / pan-ui

A chat-first workspace that integrates a marketplace for skills, MCP servers, and plugins to extend agent capabilities. It utilizes isolated profile management to keep sessions, memory, and API keys separate for different workflows.

pyrate-llama / hermes-ui

Featuring a glassmorphic design, this single-file HTML interface uses a React-based architecture to provide live previews of HTML, SVG, and PDF artifacts. It includes a built-in memory inspector and file browser for managing local agent data.
