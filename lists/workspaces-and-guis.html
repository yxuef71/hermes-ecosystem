breakdown

nesquena / hermes-webui

This lightweight, framework-free interface provides a classic three-panel layout for managing sessions, chat, and workspace files simultaneously. It maintains full parity with the Hermes CLI, offering a seamless transition for users who want a web-based or mobile-friendly experience without complex installation steps.

fathah / hermes-desktop

A comprehensive desktop companion that simplifies the setup process through guided installation for both local and remote backends. It distinguishes itself by managing 16 different messaging gateways and providing specialized tools for memory editing and task scheduling.

EKKOLearnAI / hermes-web-ui

This analytics-focused dashboard offers unified configuration for eight messaging platforms and detailed tracking of token consumption and costs. It features real-time chat streaming with markdown support and a dedicated inspector for monitoring agent tool calls.

outsourc-e / hermes-workspace

A native web workspace that integrates a terminal, memory browser, and skill manager into a single interface. It is unique for its 'Swarm Mode,' which allows users to orchestrate multiple persistent, role-based autonomous agents within a vanilla Hermes environment.

dodo-reach / hermes-desktop

Tailored specifically for macOS, this workspace uses native SSH and aliases to connect directly to Hermes hosts. It manages remote sessions and cron jobs alongside Kanban boards, supporting complex multi-agent workflows through selectable profiles.

xaspx / hermes-control-interface

This self-hosted gateway acts as a central hub for managing multiple Hermes profiles and monitoring system metrics. It provides a robust terminal and file explorer behind a secure login, complemented by detailed analytics on model usage and platform costs.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React application designed to run as a Telegram Mini App, featuring a terminal-style chat interface with slash command support. It allows users to spawn background agents and monitor live outputs or manage scheduled cron jobs directly from their mobile device.

pyrate-llama / hermes-ui

This glassmorphic web interface uses a lightweight Python proxy to deliver a single-file React application for managing self-hosted assistants. It combines real-time log streaming with an integrated file browser and management tools for MCP skills and cron jobs.

aivrar / portable-hermes-agent

A zero-install Windows desktop solution that operates without admin rights or Docker, making Hermes highly accessible. It comes pre-loaded with over 100 tools, including ComfyUI and TTS, and features a dynamic tool maker for creating custom API wrappers.

sanchomuzax / hermes-webui

Focused on process monitoring, this dashboard provides real-time visibility into gateway status and session activity. It features deep conversation inspection using FTS5 search, allowing users to audit message-level details and track costs across the ecosystem.

Euraika-Labs / pan-ui

A chat-first workspace that emphasizes extensibility through an integrated marketplace for discovering skills and MCP servers. It offers isolated profile management to keep sessions, memory, and API keys separate while providing granular runtime controls for tool approvals.
