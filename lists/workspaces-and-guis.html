breakdown

nesquena / hermes-webui

This zero-build Python and JavaScript interface provides full feature parity with the Hermes CLI in a mobile-friendly web environment. It features a streamlined three-panel layout that organizes chat, file browsing, and session management into a single cohesive view.

fathah / hermes-desktop

This desktop companion simplifies the setup process with guided installation and dependency resolution for both local and remote backends. It supports 14 distinct toolsets, including vision and code execution, while managing gateway connections for various messaging platforms.

EKKOLearnAI / hermes-web-ui

This dashboard serves as a multi-platform command center, offering unified configuration for eight different channels including Telegram, Discord, and Slack. It distinguishes itself with real-time usage analytics, token tracking, and an integrated web terminal for remote backend management.

outsourc-e / hermes-workspace

Designed as a native web workspace, this project features a 'Swarm Mode' for the role-based dispatch of multiple persistent agents. It integrates chat, memory browsing, and terminal operations into a unified dashboard that works with vanilla Hermes installations without requiring forks.

dodo-reach / hermes-desktop

This native macOS application provides a direct SSH connection to Hermes sessions without requiring a middle-tier API or daemon. It offers a unique productivity view that combines remote terminal access with Kanban boards, cron job management, and file browsing.

xaspx / hermes-control-interface

This self-hosted dashboard acts as a multi-agent gateway for starting, stopping, and configuring individual agent profiles. It includes a split-view file editor with syntax highlighting and detailed performance metrics to track model costs and token usage in real-time.

clawvader-tech / hermes-telegram-miniapp

This mobile-first React dashboard is built specifically as a Telegram Mini App, featuring a terminal-style chat interface with slash command support. It allows users to spawn up to five concurrent background agents while monitoring real-time system metrics like CPU and memory usage.

pyrate-llama / hermes-ui

Built on a single-file React architecture, this interface features a glassmorphic design with real-time log streaming and inline workspace previews. It provides a lightweight web-based management layer for handling MCP tools, skills, and automated agent tasks.

aivrar / portable-hermes-agent

This zero-install Windows GUI provides a portable environment for Hermes with granular permission controls and over 100 integrated tools, including TTS and ComfyUI. It stands out with a dynamic tool maker that allows users to create custom API wrappers at runtime without administrative rights.

sanchomuzax / hermes-webui

Focused on process monitoring and health, this dashboard provides real-time tracking of session activity, service status, and operational costs. It features full-text search for inspecting conversation histories and a dedicated interface for browsing skill sources and scheduled cron jobs.

Euraika-Labs / pan-ui

This chat-first workspace utilizes SSE streaming and tool approval cards to give users fine-grained control over agent actions. It includes an integrated marketplace for discovering MCP servers and isolated profile management to keep sessions, memory, and API keys separate.
