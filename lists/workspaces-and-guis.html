breakdown

nesquena / hermes-webui

This zero-build interface features a three-panel layout that combines chat, session management, and workspace file browsing into a single view. It is optimized for both desktop and mobile use, offering a unique circular context ring to visually track token usage in real time.

fathah / hermes-desktop

This desktop companion simplifies the setup process with a guided installer for both local and remote Hermes backends. It provides a streamlined interface for configuring 16 different messaging gateways and managing a diverse library of 14 functional toolsets.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard designed for multi-platform orchestration, supporting unified configuration for eight different channels including Slack, WhatsApp, and Discord. It provides deep visibility into agent operations through detailed tool-call logs and advanced analytics for tracking token consumption and costs.

outsourc-e / hermes-workspace

This workspace focuses on autonomous task management by integrating a Kanban board for tracking agent missions alongside a native terminal and memory browser. It stands out for its Swarm Mode, which enables multi-agent orchestration and role-based task dispatching within a unified interface.

dodo-reach / hermes-desktop

A native macOS application that provides direct access to Hermes hosts via real SSH connections and terminal aliases. It allows users to manage remote sessions, cron jobs, and multi-agent profiles natively without relying on a standard web browser.

xaspx / hermes-control-interface

Acting as a self-hosted gateway, this dashboard allows users to start, stop, and configure individual agent profiles from a central control panel. It pairs a browser-based terminal with collapsible tool-call cards and comprehensive performance metrics to monitor the entire Hermes stack.

clawvader-tech / hermes-telegram-miniapp

This mobile-first React application brings the Hermes dashboard into the Telegram ecosystem as a Mini App. It features a terminal-style chat interface with slash commands, background agent spawning, and live system health gauges for monitoring on the go.

pyrate-llama / hermes-ui

Built with a glassmorphic React design, this single-file HTML interface offers a lightweight yet modern way to interact with self-hosted agents. It integrates a file browser with inline previews directly alongside real-time logs, task boards, and MCP tool monitoring.

aivrar / portable-hermes-agent

A zero-install Windows GUI that runs without admin rights or Docker, focusing on local model integration via LM Studio. It comes pre-loaded with over 100 tools for media generation and includes a dynamic tool maker for building custom API wrappers and Python handlers.

sanchomuzax / hermes-webui

A monitoring-centric dashboard that emphasizes transparency through FTS5-powered search for conversation history and real-time cost tracking. It provides a read-only management view of critical system files, environment variables, and active gateway statuses.

Euraika-Labs / pan-ui

This chat-first workspace introduces an integrated marketplace for discovering and installing new skills and MCP servers. It prioritizes security and organization through isolated profile management and manual approval cards for agent tool executions.
