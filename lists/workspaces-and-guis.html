breakdown

nesquena / hermes-webui

This vanilla JavaScript interface offers a streamlined three-panel layout for simultaneous chat, session management, and workspace file browsing. It maintains full feature parity with the Hermes CLI and includes a unique visual context ring for real-time token usage monitoring.

fathah / hermes-desktop

This companion app simplifies the setup process through an automated guided installation for both local and remote Hermes backends. It excels at gateway management, supporting 16 different messaging platforms alongside integrated memory and persona editors.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard focused on multi-channel connectivity, allowing users to configure and manage Hermes across Telegram, Discord, Slack, and WhatsApp. It features real-time chat streaming and detailed usage analytics to track token consumption and estimated costs.

outsourc-e / hermes-workspace

This native web workspace provides a unified interface for chat, terminal access, memory editing, and skill management. It is specifically designed for 'Swarm Mode,' enabling the orchestration of multiple persistent, role-based autonomous agents within a single environment.

dodo-reach / hermes-desktop

This native macOS application connects directly to Hermes hosts via SSH to provide a high-performance terminal and session manager. It integrates productivity tools like Kanban boards and cron job management natively into the desktop experience.

xaspx / hermes-control-interface

A self-hosted gateway that allows for the management of several distinct Hermes profiles and system metrics from a single dashboard. The interface includes tool-call cards for interactive sessions and comprehensive token analytics to monitor costs across different models.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React dashboard designed to run as a Telegram Mini App, featuring a terminal-style chat interface with slash command support. It allows users to spawn background agents and monitor live outputs directly from their mobile devices.

pyrate-llama / hermes-ui

This lightweight, glassmorphic React application uses a Python proxy server to provide a modern web interface for Hermes. It combines a file browser and terminal with real-time log streaming and a dedicated browser for managing MCP tools and skills.

aivrar / portable-hermes-agent

A zero-install Windows desktop GUI that provides a portable environment for Hermes with over 100 integrated tools, including TTS and ComfyUI. It features a dynamic tool maker for creating custom API wrappers and Python handlers without requiring administrative rights.

sanchomuzax / hermes-webui

A monitoring-centric dashboard that focuses on session activity, gateway status, and full conversation history inspection using FTS5 search. It provides a read-only configuration viewer and a management interface for tracking scheduled cron jobs and costs.

Euraika-Labs / pan-ui

A chat-first workspace that utilizes SSE streaming and tool approval cards for secure agent interactions. It includes an integrated marketplace for discovering skills and MCP servers, supported by isolated profile management for separate API keys and memory.
