breakdown

farion1231 / cc-switch

A lightweight, Tauri-powered desktop manager that provides a unified interface for Hermes Agent alongside other major coding CLIs like Claude Code and Gemini. It offers cross-platform support for Windows, macOS, and Linux, making it an essential tool for developers balancing multiple AI agent environments.

iOfficeAI / AionUi

This local open-source workspace acts as a unified GUI for Hermes Agent and over 13 other CLI agents, featuring a built-in engine for generating editable office documents. It is uniquely designed for 24/7 unattended automation, utilizing Cron jobs and remote messaging to handle tasks autonomously.

nesquena / hermes-webui

A framework-free web interface that provides full parity with the Hermes CLI through a clean, three-panel layout for sessions, chat, and file browsing. Its lightweight architecture requires no build steps, making it the most accessible way to manage Hermes from a mobile or desktop browser.

fathah / hermes-desktop

A comprehensive desktop companion that simplifies the installation and configuration of Hermes Agent backends across local and remote environments. It stands out for its robust gateway management, allowing users to control the agent through 16 different messaging platforms like Telegram and Discord.

EKKOLearnAI / hermes-web-ui

This web dashboard focuses on multi-platform connectivity and operational oversight, supporting eight messaging channels including WhatsApp and Slack. It provides advanced usage analytics for tracking token costs and model distribution, paired with real-time chat streaming and tool call inspection.

outsourc-e / hermes-workspace

A native web workspace that integrates a terminal, memory browser, and skill manager into a single interface compatible with vanilla Hermes installations. It features a unique Swarm Mode for orchestrating multiple persistent, role-based autonomous agents within the same environment.

dodo-reach / hermes-desktop

A specialized macOS workspace that connects to Hermes hosts via native SSH and aliases to manage remote sessions and Kanban boards. It is tailored for power users who need to handle multi-agent workflows through selectable Hermes profiles and native cron job management.

xaspx / hermes-control-interface

A self-hosted dashboard that serves as a multi-agent gateway, offering a browser-based terminal and a comprehensive agent status panel. It excels at session resumption and financial oversight, providing detailed token analytics and cost tracking across different models.

jazzyalex / agent-sessions

A privacy-focused macOS session browser that indexes Hermes and other coding agent histories locally without telemetry. It allows users to quickly search past interactions and resume sessions directly in professional terminal emulators like iTerm2 or Warp.

nickvasilescu / hermes-desktop-os1

Designed for Orgo cloud computers and SSH hosts, this macOS workspace offers one-click automated installation of Hermes Agent on fresh cloud VMs. It integrates WebRTC voice mode and Orgo MCP, enabling a native, voice-driven interaction layer for cloud-based agents.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React dashboard designed to run as a Telegram Mini App, featuring a terminal-style chat with slash command support. It allows users to spawn background agents and monitor live outputs or scheduled cron jobs directly from their mobile device.

Eynzof / hermes-agent-cn-desktop

A Windows-optimized desktop application built with Tauri and Rust that provides a secure, production-grade proxy for Hermes Agent communication. It simplifies the core installation process and updates, offering a high-performance native experience for the Chinese-speaking community.

pyrate-llama / hermes-ui

A glassmorphic web interface that combines a lightweight Python proxy with a single-file React application for easy self-hosting. It provides an integrated file browser and real-time log streaming, alongside dedicated sections for managing MCP tools and skills.

aivrar / portable-hermes-agent

A zero-install Windows GUI that enables Hermes Agent to run without admin rights or Docker, utilizing local models via LM Studio. It comes pre-loaded with over 100 tools for TTS, music generation, and image creation, including a dynamic tool maker for custom API wrappers.

sanchomuzax / hermes-webui

This dashboard focuses on process monitoring and configuration, offering real-time visibility into session activity and gateway status. It features full conversation history inspection with FTS5 search, allowing users to audit message-level details and manage scheduled tasks.

Euraika-Labs / pan-ui

A self-hosted workspace that prioritizes runtime control and security through tool approval cards and isolated profile management. It includes an integrated marketplace for discovering skills and MCP servers, ensuring each agent session has its own distinct memory and API keys.

Felix-Forever / hermes-agent-desktop

A multi-agent desktop client that uses a Project Manager system to orchestrate 20 specialized AI agents. It features a visual Skill Store with 50+ one-click installable tools, making it a powerful hub for complex, tool-heavy agent workflows.
