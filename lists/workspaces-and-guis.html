breakdown

nesquena / hermes-webui

This lightweight Python and vanilla JavaScript interface provides full feature parity with the Hermes CLI in a mobile-friendly web format. It features a three-panel layout that allows users to browse workspace files and view inline previews alongside their chat sessions.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard focused on multi-platform connectivity, offering unified configuration for eight channels including Telegram, Discord, and WhatsApp. It includes real-time session persistence, tool call expansion, and detailed usage analytics for tracking token costs across different platforms.

outsourc-e / hermes-workspace

This native web workspace acts as a multi-agent control plane supporting role-based dispatch and swarm orchestration. It integrates a Monaco-based file manager and a dedicated memory browser, allowing users to search and edit agent context directly within the interface.

fathah / hermes-desktop

This desktop companion simplifies the setup process with a guided installer for local dependencies and environment configuration. It provides a unified GUI for managing 16 messaging gateways and 14 toolsets, featuring full-text search across all integrated session histories.

dodo-reach / hermes-desktop

This native macOS workspace utilizes direct SSH connections to ensure the remote host remains the single source of truth for all agent operations. The interface combines an embedded tabbed terminal with specialized views for Kanban boards, session data, and skill management.

xaspx / hermes-control-interface

A self-hosted gateway designed for managing individual agent profiles with the ability to start, stop, and configure them via a browser-based terminal. It features an integrated xterm.js terminal with full PTY support and detailed token analytics for monitoring model usage and system metrics.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React dashboard designed as a Telegram Mini App that supports spawning up to five concurrent agent instances. It features a terminal-style chat with slash commands and real-time system monitoring gauges for CPU, memory, and disk usage.

pyrate-llama / hermes-ui

This glassmorphic React application features a single-file architecture that streams logs and chat responses in real time. It uniquely includes an integrated Kanban board for automatic task detection and tracking based on agent interactions within the workspace.

aivrar / portable-hermes-agent

A zero-install Windows desktop GUI that requires no administrative rights and comes pre-loaded with over 100 tools for local TTS, music, and ComfyUI generation. It includes a dynamic tool maker for creating custom API wrappers at runtime and offers granular permission controls for local model execution.

sanchomuzax / hermes-webui

A monitoring-focused dashboard that provides a real-time activity feed for tracking session costs and agent processes. It allows for read-only configuration viewing and management of scheduled cron jobs, supported by FTS5 search capabilities for conversation history inspection.

Euraika-Labs / pan-ui

A chat-first workspace that utilizes SSE streaming and visual timelines for tool execution. It features an integrated marketplace for discovering MCP servers and skills, alongside profile-scoped management for global memory and agent configurations.
