breakdown

nesquena / hermes-webui

This zero-build web interface provides full feature parity with the Hermes Agent CLI through a streamlined three-panel layout for chat, file browsing, and session management. It is designed for high performance on both desktop and mobile browsers using a lightweight Python and vanilla JavaScript architecture.

outsourc-e / hermes-workspace

This security-hardened workspace offers a native web experience featuring real-time SSE streaming for chat and tool call rendering. It includes a dedicated memory browser and skill manager for agent orchestration, protected by execution approval prompts and authentication middleware.

EKKOLearnAI / hermes-web-ui

A comprehensive dashboard that centralizes multi-platform configuration for Telegram, Discord, Slack, and WhatsApp alongside real-time token and cost analytics. It features a versatile multi-backend file browser that supports local, Docker, SSH, and Singularity environments.

dodo-reach / hermes-desktop

A native macOS workspace that facilitates direct SSH connections to Hermes agents without requiring remote helper services. It features a multi-tabbed terminal with live theme controls and profile-aware management for sessions, skills, and cron jobs.

xaspx / hermes-control-interface

Acting as a multi-agent gateway, this self-hosted dashboard allows users to start, stop, and configure multiple agent profiles from a single interface. It provides detailed token analytics and a browser-based terminal to monitor system metrics and agent status in real-time.

fathah / hermes-desktop

A desktop companion app designed to simplify the initial setup of Hermes Agent through guided installation and dependency resolution. It supports a wide range of LLM providers and local endpoints while managing messaging gateways for platforms like Discord.

clawvader-tech / hermes-telegram-miniapp

This mobile-first React SPA brings a terminal-style chat experience to the Telegram Mini App ecosystem, supporting streaming responses and slash commands. It allows for background agent spawning with live output monitoring and integrated system health gauges.

Web3CZ / Web3Hermes

This localized Chinese interface provides a tailored experience for regional users with dark and light theme support and comprehensive session export features. It includes an integrated workspace file browser for direct editing and visualization of tool calls.

sanchomuzax / hermes-webui

Focused on process monitoring, this dashboard provides real-time activity tracking and cost distribution analysis for Hermes sessions. It features FTS5 search capabilities for inspecting conversation history and a read-only interface for managing scheduled cron jobs.

Euraika-Labs / pan-ui

A chat-first workspace that emphasizes tool execution timelines and SSE streaming for a transparent agent experience. It features an integrated marketplace for skills, MCP servers, and plugins, alongside isolated profile management for separate memory and keys.

pyrate-llama / hermes-ui

This glassmorphic web interface uses a unique single-file React architecture that requires no npm dependencies for deployment. It includes a real-time artifact panel for live HTML, SVG, and PDF previews, alongside management tools for MCP extensions and memory files.
