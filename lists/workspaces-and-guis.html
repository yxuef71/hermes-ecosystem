breakdown

nesquena / hermes-webui

This zero-build web interface offers full parity with the Hermes CLI through a streamlined three-panel layout for chat, sessions, and file browsing. It is designed for maximum accessibility, allowing users to manage persistent memory and workspace files from any browser or mobile device without complex installation.

fathah / hermes-desktop

A comprehensive desktop companion that simplifies the lifecycle of Hermes Agent through automated guided installations for local or remote backends. It distinguishes itself by managing 16 different messaging gateways, including Telegram and Slack, alongside integrated tools for task scheduling and persona configuration.

EKKOLearnAI / hermes-web-ui

This dashboard focuses on multi-platform connectivity and operational oversight, providing unified configuration for major chat channels like WhatsApp and Discord. It features real-time chat streaming and a dedicated analytics suite for tracking token consumption and estimated costs across different sessions.

outsourc-e / hermes-workspace

A native web workspace that introduces Swarm Mode for orchestrating multiple persistent, role-based autonomous agents from a single interface. It provides a unified environment for chat, terminal access, and skill management while maintaining zero-fork compatibility with vanilla Hermes installations.

dodo-reach / hermes-desktop

Built specifically for macOS, this workspace utilizes native SSH and aliases to connect directly to Hermes hosts for a high-performance terminal experience. It integrates remote session management with Kanban boards and cron jobs, supporting complex multi-agent workflows through selectable profiles.

xaspx / hermes-control-interface

This self-hosted gateway acts as a central command center for managing multiple Hermes profiles and monitoring system metrics in real-time. It features a specialized chat interface with tool-call cards and comprehensive token analytics to track usage across different models and platforms.

clawvader-tech / hermes-telegram-miniapp

A mobile-first React application designed to run as a Telegram Mini App, offering a terminal-style chat interface with slash command support. It enables users to spawn background agents and monitor live outputs or manage scheduled cron jobs directly from within the Telegram ecosystem.

pyrate-llama / hermes-ui

This lightweight, glassmorphic web interface uses a single-file React application and a Python proxy to provide a modern aesthetic for self-hosted assistants. It combines a file browser and terminal with specialized management for Model Context Protocol (MCP) tools and real-time log streaming.

aivrar / portable-hermes-agent

A zero-install Windows desktop solution that requires no admin rights or Docker, making Hermes highly portable. It comes pre-loaded with over 100 tools—including TTS and ComfyUI—and features a dynamic tool maker for creating custom API wrappers and Python handlers.

sanchomuzax / hermes-webui

Focused on observability, this dashboard provides a real-time overview of session activity, gateway status, and cost tracking. It features deep conversation inspection using FTS5 search, allowing users to audit message-level details and manage scheduled cron jobs through a read-only configuration viewer.

Euraika-Labs / pan-ui

A chat-first workspace that emphasizes extensibility through an integrated marketplace for discovering and installing new skills and MCP servers. It provides isolated profile management to keep sessions, memory, and API keys separate while utilizing SSE streaming for real-time tool approval cards.
