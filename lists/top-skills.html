breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to major industry frameworks like MITRE ATT&CK and NIST. It enables Hermes agents to perform sub-second discovery and execution across 26 security domains using the agentskills.io standard.

Agents365-ai / drawio-skill

This skill allows agents to generate complex draw.io diagrams, including UML and deep learning architectures, from natural language descriptions. It features a visual feedback loop that automatically detects and fixes layout issues before exporting to formats like PNG or SVG.

conorbronsdon / avoid-ai-writing

Designed to eliminate detectable AI patterns, this tool uses a two-pass audit and a 42-category pattern library to rewrite content. It helps Hermes agents produce more human-like text by utilizing tiered vocabulary tables and specialized detection modes.

AMAP-ML / SkillClaw

SkillClaw introduces an 'Agentic Evolver' that automates skill deduplication and quality improvement through background evolution loops. It unifies skill libraries across multiple devices and teams, offering native integration with Hermes and OpenAI-compatible APIs.

wondelai / skills

This is a modular, cross-platform library providing specialized frameworks for architecture, UX design, and product strategy. It is fully compatible with Claude Code and agentskills.io, allowing for easy installation via a dedicated CLI.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose reusable skill patterns. It streamlines development by generating both SKILL.md documentation and Python scaffolds directly from observed agent actions.

DougTrajano / pydantic-ai-skills

This project implements the Agent Skills open specification for Pydantic AI with a focus on type safety. It utilizes progressive disclosure of metadata to significantly reduce token usage during skill discovery.

amanning3390 / hermeshub

HermesHub serves as a community marketplace for browsing and installing skills, featuring automated security scanning with over 65 threat detection rules. It includes an agent-to-agent feedback system and supports monetization through crypto and Stripe payments.

Cranot / super-hermes

This project teaches Hermes agents to write their own analytical prompts and identify structural conservation laws within complex codebases. It generates constraint reports that highlight potential analysis blind spots and limitations in the agent's reasoning.

ReinaMacCredy / maestro

Maestro is a harness for long-running agents that manages shared state locally without requiring a database. It decomposes complex specifications into multi-task execution plans and uses JSON handoff envelopes for seamless transitions between sessions.

tlehman / litprog-skill

This literate programming skill enables agents to weave prose, diagrams, and source code into unified .lit.md files. It includes a tangler to extract runnable code and automatically syncs edits between source files and their documentation.

esaradev / icarus-plugin

Icarus provides a shared memory system via markdown-based fabric directories, allowing agents to remember past work across different instances. It features an end-to-end pipeline for extracting decision-quality data to fine-tune replacement models.

smartcontractkit / chainlink-agent-skills

This library provides standardized skills for interacting with Chainlink oracles, including CCIP, VRF, and Data Feeds. It follows the agentskills.io spec to facilitate secure smart contract onboarding and local workflow simulations.

armelhbobdad / bmad-module-skill-forge

Skill Forge converts existing code repositories and developer documentation into verified, agentskills.io-compliant instructions. It ensures high reliability by providing direct traceability back to source file line numbers.

black-forest-labs / skills

This official repository provides standardized prompting guidelines and API integration patterns for the FLUX image generation model family. It enables Hermes to handle precise hex color specifications and structured JSON prompting for high-fidelity visual outputs.

chigwell / skilldock.io

SkillDock is a registry for reusable AI skills that supports automated installation with recursive dependency resolution. It features a Python SDK for discovery and offers monetization options via Stripe and the TON blockchain.

tiann / execplan-skill

This skill enables autonomous execution of multi-step tasks with built-in checkpoints for real-time progress tracking. It ensures operational safety by providing rollback procedures and recovery mechanisms for long-running agent processes.

cablate / Agentic-MCP-Skill

This progressive MCP client uses a three-layer lazy loading architecture to reduce token usage by up to 86%. It maintains persistent server connections via a socket-based daemon and supports hot-reloading of configurations.

PederHP / skillsdotnet

This C# and .NET implementation of the agentskills.io specification brings tool-based skill loading to the Microsoft.Extensions.AI ecosystem. It supports progressive disclosure via SKILL.md files and automatic registration of common skill directories.

adnw-vinc / hermes-nextcloud

This skill integrates Hermes with Nextcloud, allowing the agent to manage files, calendars, and contacts via WebDAV and CalDAV protocols. It ensures secure communication over HTTPS and includes a guided setup script for credential validation.
