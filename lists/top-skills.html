breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to industry standards like MITRE ATT&CK and NIST CSF 2.0. It enables Hermes to perform specialized tasks across 26 domains, including malware analysis and cloud security, using the agentskills.io standard.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate detectable AI patterns using a tiered 109-entry word replacement table. It ensures Hermes's output remains human-like through a two-pass audit process compatible with Claude Code and OpenClaw.

wondelai / skills

A modular cross-platform library that grants agents capabilities in architecture, UX design, and product strategy. It features a dedicated CLI for easy installation and full compatibility with the agentskills.io ecosystem.

Agents365-ai / drawio-skill

This skill allows Hermes to generate six types of professional diagrams, including UML and ML architectures, from natural language. It includes a five-round self-check safety valve to iteratively refine visual layouts and coordinate snapping.

DougTrajano / pydantic-ai-skills

This project implements the Agent Skills open specification for Pydantic AI with a focus on type safety. It utilizes progressive disclosure of metadata to significantly reduce token usage during skill discovery.

Romanescu11 / hermes-skill-factory

This meta-skill monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and documentation, allowing Hermes to expand its own capabilities via slash commands.

tlehman / litprog-skill

This literate programming tool allows Hermes to weave source code and prose into unified .lit.md files. It includes a tangler to extract runnable code and automatically syncs edits between documentation and source files.

esaradev / icarus-plugin

Icarus provides Hermes with shared memory across instances using markdown-based fabric directories. It automates the extraction of decision-quality training data to help the agent fine-tune its own replacement models.

smartcontractkit / chainlink-agent-skills

Designed for blockchain interoperability, this skill enables Hermes to perform cross-chain sends via CCIP and monitor multi-chain data feeds. It follows the standardized agentskills.io spec to facilitate secure smart contract interactions.

ReinaMacCredy / maestro

Maestro is a local-first harness for long-running tasks that utilizes structured mission planning and JSON snapshots. It enables Hermes to manage complex features through a plan-approve-execute workflow with native handoff briefs.

chigwell / skilldock.io

This registry manages the lifecycle of reusable AI skills through an OpenAPI-driven Python SDK and CLI. It uniquely supports skill monetization via Stripe and the TON blockchain while handling automated dependency resolution.

black-forest-labs / skills

The official skill set for FLUX image generation, providing Hermes with precise prompting guidelines and API integration patterns. It supports structured JSON prompting and exact hex color specifications for high-fidelity visual outputs.

tiann / execplan-skill

This skill focuses on the autonomous execution of multi-step tasks with built-in checkpoints for real-time progress tracking. It ensures reliability by providing rollback procedures for safe recovery during complex operations.

amanning3390 / hermeshub

A community hub for browsing and installing skills that features automated security scanning for prompt injection. It supports micropayment protocols for premium content and includes an agent-to-agent feedback system with trust scores.

armelhbobdad / bmad-module-skill-forge

This forge converts repositories and developer discourse into compliant AI skills by mapping instructions to specific source code lines. it allows Hermes to generate version-pinned instruction sets for various software libraries.

cablate / Agentic-MCP-Skill

A progressive MCP client that uses three-layer lazy loading to minimize context token consumption. It maintains persistent server connections via a socket-based daemon and supports hot-reloading of configurations.

PederHP / skillsdotnet

This project brings the agentskills.io specification to the .NET ecosystem using the C# MCP SDK. It supports automatic skill discovery and progressive disclosure to maintain efficiency in C#-based agent environments.
