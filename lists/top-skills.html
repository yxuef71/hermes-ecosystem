breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to major frameworks like MITRE ATT&CK and NIST. It enables Hermes to perform sub-second AI skill discovery across 26 specialized security domains using the agentskills.io standard.

Agents365-ai / drawio-skill

This skill allows agents to generate complex draw.io diagrams from natural language descriptions with built-in presets for UML and ML models. It includes a self-check visual feedback loop to automatically fix layout issues during the export process.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate detectable AI writing patterns by identifying 42 distinct structural tells. It utilizes a tiered word replacement table and a second-pass audit to ensure natural, plain-language output.

AMAP-ML / SkillClaw

SkillClaw uses an agentic evolver to automate skill deduplication and quality improvement based on interaction history. It unifies skill libraries across multiple devices and natively supports Hermes and Claude Code integrations.

wondelai / skills

A modular, cross-platform library that provides Hermes with frameworks for architecture, UX design, and product strategy. It features a dedicated CLI for easy installation and full compatibility with agentskills.io-compliant agents.

Romanescu11 / hermes-skill-factory

This meta-skill plugin monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and SKILL.md documentation, allowing the agent to expand its own capabilities via slash commands.

DougTrajano / pydantic-ai-skills

This project implements the Agent Skills open specification for Pydantic AI with a focus on type safety. It reduces token consumption through progressive disclosure of skill metadata and supports both filesystem and programmatic definitions.

amanning3390 / hermeshub

Hermeshub is a community marketplace for browsing and installing skills with integrated security scanning and threat detection. It features an agent-to-agent feedback system and supports creator payments via crypto or Stripe.

Cranot / super-hermes

Super-hermes teaches agents to write their own analytical prompts and custom thinking instructions for complex problem-solving. It is designed to identify structural conservation laws in codebases and generate reports on analysis blind spots.

ReinaMacCredy / maestro

Maestro is a local-first harness that manages long-running agent tasks using on-disk state files instead of chat history. It facilitates a structured spec-to-ship workflow and uses passive handoff envelopes for seamless task resumption.

tlehman / litprog-skill

This skill implements literate programming by weaving source code, prose, and diagrams into unified .lit.md files. It features a tangler to extract runnable code and automatically syncs edits between source files and documentation.

esaradev / icarus-plugin

Icarus provides cross-instance shared memory via markdown-based directories to help agents remember their work over time. It includes an end-to-end pipeline for extracting decision-quality training data to fine-tune replacement models.

smartcontractkit / chainlink-agent-skills

This library enables Hermes to interact with decentralized oracle networks for CCIP, VRF, and data feed operations. It supports cross-chain identity management and automated proof of reserve verification within agent-led workflows.

armelhbobdad / bmad-module-skill-forge

Skill-forge converts repositories and developer discourse into version-pinned, agentskills.io-compliant instructions. It automates the creation process through Brief, Create, and Test workflows while maintaining verifiable file-and-line provenance.

black-forest-labs / skills

These official skills provide standardized prompting guidelines and API integration patterns for the FLUX image generation model family. They enable Hermes to use precise hex color specifications and structured JSON prompting for high-fidelity visual output.

chigwell / skilldock.io

Skilldock is a registry for reusable AI skills that features an OpenAPI-driven SDK and CLI for lifecycle management. It uniquely supports skill monetization through Stripe and the TON blockchain while handling automated dependency resolution.

tiann / execplan-skill

This skill focuses on the autonomous execution of complex, multi-step tasks with integrated checkpoints. It provides real-time progress tracking and built-in rollback procedures to ensure safe recovery from execution failures.

cablate / Agentic-MCP-Skill

This progressive MCP client uses a three-layer lazy loading pattern to significantly reduce context token usage. It maintains persistent server connections via a socket-based daemon and supports hot-reloading for dynamic configuration updates.

PederHP / skillsdotnet

This C# and .NET implementation of the Agent Skills specification integrates with Microsoft.Extensions.AI for tool-based loading. It supports progressive disclosure via SKILL.md files and provides automatic registration for common skill directories.

adnw-vinc / hermes-nextcloud

This skill enables Hermes to manage files, calendars, tasks, and contacts through standard WebDAV and CalDAV protocols. It ensures secure communication via Nextcloud App Passwords and includes a guided setup script for easy configuration.
