breakdown

mukul975 / Anthropic-Cybersecurity-Skills

This repository provides 754 structured cybersecurity skills mapped to industry standards like MITRE ATT&CK and NIST CSF 2.0. It enables Hermes to perform specialized tasks across 26 domains, including malware analysis and cloud security, using the agentskills.io standard.

conorbronsdon / avoid-ai-writing

This tool audits and rewrites content to eliminate detectable AI patterns using a three-tier word replacement system. It ensures Hermes-generated text maintains a human-like quality through a rigorous two-pass refinement process.

wondelai / skills

A modular cross-platform library that equips agents with frameworks for architecture, UX design, and product strategy. It features a dedicated CLI for easy installation and full compatibility with the agentskills.io ecosystem.

Agents365-ai / drawio-skill

This skill allows Hermes to generate six types of professional diagrams, including UML and ML architectures, from natural language descriptions. It includes a five-round self-check safety valve to ensure coordinate precision and iterative refinement.

DougTrajano / pydantic-ai-skills

This project implements type-safe support for the Agent Skills specification within the Pydantic AI framework. It optimizes Hermes' performance by using progressive disclosure of metadata to significantly reduce token consumption.

Romanescu11 / hermes-skill-factory

A meta-skill plugin that monitors active workflows to automatically detect and propose new, reusable skills. It generates both Python scaffolds and documentation, allowing Hermes to expand its own capabilities through observation.

tlehman / litprog-skill

This literate programming tool allows Hermes to weave runnable code and prose together into synchronized markdown documents. It features a 'tangler' to extract source files and automatically updates documentation when code edits are detected.

esaradev / icarus-plugin

This plugin enables cross-instance shared memory, allowing Hermes to remember past work and extract decision-quality data for model fine-tuning. It creates an end-to-end pipeline for an agent to document its own logic and train its eventual replacement.

smartcontractkit / chainlink-agent-skills

Designed for blockchain interoperability, this skill enables Hermes to interact with Chainlink Data Feeds and execute cross-chain transfers via CCIP. It provides a standardized way for agents to monitor smart contracts and manage local testing workflows.

ReinaMacCredy / maestro

This harness manages long-running missions through structured planning, milestones, and local-first state snapshots. It facilitates seamless handoffs between agent runs by generating detailed markdown briefs and validation assertions.

chigwell / skilldock.io

A comprehensive registry and SDK for managing the lifecycle of reusable AI skills with built-in monetization via Stripe and blockchain. It automates dependency resolution and provides a CLI for local installation of OpenAPI-driven skills.

black-forest-labs / skills

This official integration provides standardized prompting guidelines and API patterns for the FLUX image generation model family. It allows Hermes to handle precise hex color specifications and structured JSON prompting for high-fidelity visual outputs.

amanning3390 / hermeshub

A community-driven hub for browsing and installing Hermes skills with integrated security scanning for prompt injection. It supports micropayment protocols for premium content and features an agent-to-agent feedback system for trust scores.

tiann / execplan-skill

This skill provides Hermes with a framework for autonomous execution of complex, multi-step tasks. It features real-time progress tracking and built-in rollback procedures to ensure safe recovery from execution errors.

armelhbobdad / bmad-module-skill-forge

This utility converts existing repositories and developer documentation into agentskills.io-compliant instruction sets. It maps AI instructions directly to specific source code lines, ensuring high-precision skill generation from legacy codebases.

cablate / Agentic-MCP-Skill

A progressive MCP client that utilizes a three-layer lazy loading pattern to minimize context window usage. It maintains persistent server connections via a socket-based daemon and supports hot-reloading for dynamic configuration updates.

PederHP / skillsdotnet

This project brings the agentskills.io specification to the .NET ecosystem using the C# MCP SDK. It enables automatic skill discovery and utilizes progressive disclosure to maintain efficient token usage in C#-based agent environments.
