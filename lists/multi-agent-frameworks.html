breakdown

builderz-labs / mission-control

This self-hosted orchestration platform manages multi-agent workflows across frameworks like CrewAI and AutoGen using a zero-dependency SQLite backend. It integrates the Aegis review system for human-in-the-loop sign-offs, making it a robust choice for monitoring spend and task dispatching in production environments.

swarmclawai / swarmclaw

Swarmclaw enables the creation of autonomous agent swarms with support for over 23 LLM providers, including local execution via Ollama. It distinguishes itself through durable agent memory and a unique conversation-to-skill learning system that allows agents to evolve their capabilities during multi-agent delegation.

1ilkhamov / opencode-hermes-multiagent

This framework coordinates 17 specialized agents across six functional domains to automate complex research and implementation pipelines. It ensures high-integrity development by enforcing mandatory quality and security gates for every code change routed through the system.

howdymary / hermes-agent-metaharness

Focused on meta-optimization, this framework uses an automated outer-loop search to refine agent harness code for the Hermes model. It orchestrates TBLite and TB2 benchmark evaluations to track performance frontiers through structured candidate mutation and comparison.

runtimenoteslabs / gladiator

Gladiator creates a competitive arena where two autonomous AI companies, powered by Hermes Agent, compete for GitHub stars. The system orchestrates nine specialized agents and visualizes their real-time task execution, code generation, and scoreboard metrics on a live dashboard.

Rainhoole / hermes-agent-acp-skill

This project introduces ACP-style delegation to route tasks between Hermes subagents, Codex, and Claude Code while preventing parent context pollution. It prioritizes safety and efficiency through isolated workflows, configurable timeouts, and strict output character limits.

supermodeltools / bigiron

Bigiron implements an AI-native software development lifecycle by providing Hermes with structural code graphs via the Supermodel MCP. It executes an autonomous eight-phase workflow that enforces architectural quality through automated graph-diff gates at every stage of production.
