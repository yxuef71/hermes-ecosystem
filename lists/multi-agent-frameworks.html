breakdown

builderz-labs / mission-control

This self-hosted orchestration platform manages multi-agent workflows across frameworks like CrewAI and AutoGen using a zero-dependency SQLite backend. It is essential for Hermes users requiring strict oversight, featuring the Aegis review system for human-in-the-loop task sign-off and integrated spend monitoring.

swarmclawai / swarmclaw

Swarmclaw enables the creation of autonomous agent swarms with support for over 20 model providers and specialized features like scheduled tasks and memory. It distinguishes itself by providing visual organizational charts to manage agent delegations and team hierarchies effectively.

howdymary / hermes-agent-metaharness

This framework focuses on meta-optimization by using an automated outer-loop search to refine harness code and improve benchmark performance. It orchestrates Hermes Agent through structured candidate mutations to track and push the performance frontiers of TBLite and TB2 evaluations.

1ilkhamov / opencode-hermes-multiagent

This project coordinates 17 specialized agents across distinct development phases ranging from research to infrastructure. It ensures high-standard outputs by implementing mandatory quality gates for code reviews and testing across multiple LLM providers.

runtimenoteslabs / gladiator

Gladiator creates a competitive arena where two autonomous AI companies, powered by Hermes Agent, compete for GitHub stars. The system orchestrates nine specialized agents and provides a real-time dashboard to visualize task execution, code generation, and live scoreboard metrics.

supermodeltools / bigiron

Bigiron implements an AI-native software development lifecycle by combining Hermes Agent with Supermodel code graphs. It enforces architectural integrity through an eight-phase process where each stage is protected by automated graph-diff phase gates.

Rainhoole / hermes-agent-acp-skill

This tool utilizes ACP-style delegation to route tasks between Hermes subagents, Codex, and Claude Code while preventing context pollution. It prioritizes execution safety by enforcing configurable timeouts and strict output character limits for every delegated workflow.
