breakdown

builderz-labs / mission-control

A self-hosted orchestration platform that manages multi-agent workflows via SQLite with zero external dependencies. It supports multiple gateways like CrewAI and LangGraph and features the Aegis system for human-in-the-loop sign-offs.

agent-of-empires / agent-of-empires

A TUI and web-based control surface for monitoring sessions across various coding agents, including Hermes. It utilizes Docker, Podman, and Apple Containers for sandboxing and enables parallel execution across git worktrees.

CryptoDmitry / hermes-agent-control-room

A control-room template designed to scale Hermes agents from a single VPS to orchestrated specialist teams. It implements task bus patterns to facilitate structured delegation between orchestrators and specialists.

Agent-3-7 / minions

A management interface for Hermes agents featuring a Kanban board for tracking task status and review queues. It provides live streaming of agent reasoning and requires human verification for all proposed completions.

swarmclawai / swarmclaw

A framework for building autonomous AI swarms with durable memory and a visual protocol builder for designing workflows. It supports over 23 LLM providers, including Hermes and local Ollama instances.

shannhk / hermes-agent-control-room

A scaling template for Hermes Agent that uses a level-based architecture to transition from single agents to automated teams. It includes bundled skills for security auditing, backup management, and VPS provisioning.

1ilkhamov / opencode-hermes-multiagent

A coordinated system of 17 specialized agents across six functional domains. It automates task routing through research and implementation pipelines while enforcing mandatory security and quality gates.

linke-ai / hermes-agent-team

A local collaboration system that orchestrates Leader and Specialist agents using MCP and ACP protocols. It features a real-time Web UI for monitoring agent dialogues and Kanban task flows.

howdymary / hermes-agent-metaharness

A meta-optimization harness that uses automated outer-loop search to improve agent code. It orchestrates TBLite and TB2 benchmark evaluations to track performance frontiers through candidate mutation.

runtimenoteslabs / gladiator

An agent-vs-agent arena where autonomous AI companies compete for GitHub stars. It orchestrates nine agents using Paperclip and Hermes Agent, visualizing progress via a real-time scoreboard and task dashboard.

Rainhoole / hermes-agent-acp-skill

An ACP-style delegation tool that routes tasks between Hermes subagents, Codex, and Claude Code. It prevents parent context pollution through isolated workflows and enforces safety via output limits and timeouts.

MorvaniLabs / cordn8

A local-first coordination layer that uses an append-only markdown channel for auditable agent communication. It offers three distinct autonomy modes and universal adapters for Hermes, NVIDIA NIM, and other major model providers.
