breakdown

builderz-labs / mission-control

This self-hosted orchestration platform manages multi-agent workflows using a zero-dependency SQLite backend and a human-in-the-loop review system. It serves as a unified gateway for dispatching tasks across CrewAI, LangGraph, and AutoGen while monitoring resource spend.

agent-of-empires / agent-of-empires

This control surface manages parallel coding sessions for Hermes and other models using tmux for persistence and Docker for isolated sandboxing. It provides a versatile interface across TUI, web, and mobile to automate git worktrees and agent recovery.

CryptoDmitry / hermes-agent-control-room

Designed as a sidecar control plane, this template uses task bus patterns to scale Hermes from a single VPS agent into orchestrated specialist teams. It focuses on governance and structured delegation to ensure organized multi-agent coordination.

Agent-3-7 / minions

This mission control interface features a Kanban board for tracking autonomous Hermes Agent tasks and live-streaming their reasoning processes. It emphasizes human-in-the-loop verification, requiring manual sign-off for all agent-proposed task completions.

swarmclawai / swarmclaw

Swarmclaw enables the creation of autonomous agent swarms with durable memory and a visual protocol builder for designing complex workflows. It supports over 23 LLM providers, allowing Hermes to collaborate seamlessly with local and cloud-based models.

shannhk / hermes-agent-control-room

This framework utilizes a level-based architecture to scale Hermes agents into automated teams capable of VPS provisioning and security auditing. It implements a task bus pattern to facilitate efficient handoffs between orchestrators and specialists.

1ilkhamov / opencode-hermes-multiagent

This project coordinates 17 specialized agents across six functional domains to automate research, implementation, and infrastructure tasks. It enforces mandatory quality and security gates to ensure all code changes meet predefined standards.

linke-ai / hermes-agent-team

Built on MCP and ACP protocols, this local web system orchestrates Leader and Specialist agents for automated task lifecycle management. It features a real-time UI for monitoring agent dialogues and managing task flows via a Hermes-integrated Kanban board.

howdymary / hermes-agent-metaharness

This meta-optimization harness uses an automated outer-loop search to refine agent code and track performance frontiers. It orchestrates benchmark evaluations through Hermes, utilizing structured candidate mutation to improve agent efficiency.

runtimenoteslabs / gladiator

Gladiator creates a competitive arena where two autonomous AI companies, powered by Hermes and Paperclip, vie for GitHub stars. The system orchestrates nine agents and visualizes their tasks, code, and scoreboard metrics on a real-time dashboard.

Rainhoole / hermes-agent-acp-skill

This delegation tool routes tasks between Hermes, Codex, and Claude Code while preventing context pollution through isolated workflows. It ensures operational safety by enforcing configurable timeouts and strict output character limits.

supermodeltools / bigiron

Bigiron implements an autonomous eight-phase SDLC that uses Supermodel MCP to provide Hermes with structural code graphs. It enforces architectural quality by requiring automated graph-diff gates at every phase from planning to health checks.

MorvaniLabs / cordn8

This local-first coordination layer uses an append-only markdown channel to maintain an auditable trail of agent interactions. It offers three autonomy modes, ranging from manual approval to autopilot, for managing Hermes and other model adapters.
