overview

Web3Hermes is a lightweight WebUI for Hermes Agent specifically localized for Chinese-speaking users. It provides a desktop-optimized interface that simplifies interaction with the agent through a localized workflow and full Chinese language support. The platform integrates a built-in workspace file browser, session management tools, and support for multiple AI models including OpenAI, Anthropic, and DeepSeek. Users can manage tasks, edit code with syntax highlighting, and visualize tool calls within a responsive dark or light themed environment.

  • Localized Chinese interface with dark and light theme support
  • Integrated workspace file browser for direct file editing and management
  • Comprehensive session management including export, search, and tool call visualization

快速开始

环境要求

  • WSL 或 Linux 系统

安装步骤

# 1. 克隆仓库
git clone https://github.com/Web3CZ/Web3Hermes.git
cd Web3Hermes

# 2. 启动服务（自动安装依赖）
python3 bootstrap.py

或使用启动脚本：

./start.sh

服务将在 http://127.0.0.1:8787 启动。

配置环境变量

创建 .env 文件或设置环境变量：

# 基础配置
export HERMES_WEBUI_HOST=127.0.0.1      # 监听地址
export HERMES_WEBUI_PORT=8787           # 端口号
export HERMES_WEBUI_PASSWORD=your-pass  # 访问密码（推荐设置）

# 可选配置
export HERMES_WEBUI_STATE_DIR=~/.hermes/webui-mvp
export HERMES_WEBUI_DEFAULT_WORKSPACE=~/workspace
export HERMES_WEBUI_DEFAULT_MODEL=openai/gpt-4o-mini
