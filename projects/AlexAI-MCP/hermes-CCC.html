overview

hermes-CCC is a port of the Hermes Agent framework specifically designed for the Claude Code native skill system. It integrates the Hermes brain architecture directly into the Claude Code execution context, eliminating the need for external processes or OAuth authentication. The project implements 46 core skills across categories such as MLOps, software development, and research, including specialized tools for RL training and vector database management. This allows developers to utilize Hermes-style memory, personas, and task routing while remaining within a unified terminal environment.

  • Port of Hermes Agent brain and 46 core skills to CCC
  • Runs natively in Claude Code without separate processes or OAuth
  • Includes MLOps tools for vLLM, GGUF inference, and RL training

hermes-CCC

Hermes Agent, faithfully ported to Claude Code Channel (CCC).

hermes-CCC brings the power of Hermes Agent by NousResearch into Claude Code's native skill system — no separate process, no OAuth, no lock-in. Everything runs inside Claude Code.

Built by AlexAI-MCP.

Install

git clone https://github.com/AlexAI-MCP/hermes-CCC
cd hermes-CCC
chmod +x install.sh && ./install.sh

Windows (PowerShell):

git clone https://github.com/AlexAI-MCP/hermes-CCC
cd hermes-CCC
.\install.ps1

One-liner (Unix):

curl -fsSL https://raw.githubusercontent.com/AlexAI-MCP/hermes-CCC/main/install.sh | bash

Restart Claude Code after install to activate all skills.

Relationship to Hermes Agent

This is a Claude Code-native reinterpretation of Hermes operational patterns, not a byte-for-byte clone. The key differences:

  • No separate process — everything runs inside Claude Code
  • No OAuth — uses Claude Code's native auth
  • No gateway — uses existing Discord/Telegram MCP plugins
  • No model dependency — works with any Claude model

See docs/migration-guide.md for migration guidance from Hermes Agent.
