overview

This project provides a Termux-based installer and configuration guide for deploying the Nous Research Hermes Agent on Android devices. It automates the setup of a Linux environment via proot-distro to run the Python-based agent framework and its persistent memory systems locally. The installer enables mobile access to over 70 built-in tools and supports integration with both cloud-based APIs and local models via Ollama.

  • One-line automated installation script for Termux environments
  • Supports local model execution using Ollama on Android
  • Enables persistent memory and cross-platform tool execution

🚀 One-Line Installation

Copy and paste this command in Termux:

curl -fsSL https://raw.githubusercontent.com/AbuZar-Ansarii/Hermes-Agent-On-Android/main/nous_agent.sh | bash

🛠️ Manual Installation (Recommended)

Prefer to do it yourself? Here's the step-by-step:

pkg install git

# 1. Clone this repository
git clone https://github.com/AbuZar-Ansarii/Hermes-Agent-On-Android.git
cd Hermes-Agent-On-Android

# 2. Make the script executable
chmod +x agent_install.sh

# 3. Run the installer
./agent_install.sh

🤖 Start Agent

Run these commands one by one after installling

cd
proot-distro login ubuntu

cd hermes-agent
source venv/bin/activate

Run for setting it up

hermes setup

Run for using it

hermes

Start gateway

hermes gateway
