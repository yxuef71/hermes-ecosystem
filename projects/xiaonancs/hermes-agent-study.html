overview

This project provides a comprehensive technical analysis and documentation series focused on the internal architecture of the Hermes Agent by Nous Research. It breaks down the system's core components, including its Python-based execution loop, tool registry, and self-evolution engine, while providing nearly 100 architectural diagrams. The research also explores the project's lineage by comparing its structure to OpenClaw and investigating claims regarding its EvoMap-inspired self-improvement logic. Additionally, it tracks rapid version iterations from v0.9.0 to v0.14.0, covering new features like the Transport ABC and autonomous curators.

  • Deep source-code analysis of Hermes Agent architecture and evolution engine
  • Comparative studies against OpenClaw, EvoMap, and other Agent frameworks
  • Detailed documentation of memory systems, skill hubs, and tool registries

引言

Hermes Agent 是 2026 年最受关注的开源 AI Agent 项目之一——90k Stars，Nous Research 出品，口号是"The agent that grows with you"。

但它的火爆背后有两个绕不开的问题：

  1. 整体架构（Gateway、Tool Registry、Skill System、启动文件体系）与 OpenClaw（358k Stars）存在大量可比结构，只是从 TypeScript 换成了 Python，还内置了 hermes claw migrate 迁移工具。
  2. 核心差异化卖点"Self-evolve"被 EvoMap 指控架构级抄袭——10 步主循环对齐、12 组术语替换、三层记忆对应、7 份材料零引用。

本研究对 Hermes Agent 从三部分进行源码层面拆解：
Part I 使用方法与原理分析（11 章）
Part II 源码解析（14 章）
Part III 同源项目对比分析（7 章）
同时也针对性补充了近来社区讨论迅速升温的话题：AutoResearch。
