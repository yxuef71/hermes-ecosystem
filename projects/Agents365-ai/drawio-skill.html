overview

The drawio-skill project is a tool that converts natural-language descriptions into professional diagrams using the .drawio XML format. It utilizes the native draw.io desktop CLI to generate and export diagrams into PNG, SVG, PDF, or JPG formats. The system features six specialized presets for architecture, machine learning, and UML, while incorporating a multi-round self-check and feedback loop to refine visual accuracy. It integrates with various AI agents including Hermes, Cursor, and Claude Code to automate technical documentation workflows.

  • Includes presets for ERD, UML, Architecture, and ML diagrams

drawio-skill — From Text to Professional Diagrams

中文文档 | Online Docs

A skill that turns natural-language descriptions into .drawio XML and exports them to PNG / SVG / PDF / JPG via the native draw.io desktop CLI — with 6 diagram presets (ERD, UML Class, Sequence, Architecture, ML/DL, Flowchart), self-check + auto-fix (2 rounds), an iterative feedback loop (5 rounds), and style presets you can capture from a sample file or image.

Works with Claude Code, Cursor, Copilot, OpenClaw, Codex, Hermes, and any agent that supports the Agent Skills format.

Quick Start

Two steps — install the draw.io CLI first (see docs/INSTALL_CLI.md), then drop the skill into your host (see docs/INSTALL_SKILL.md). After that, just describe what you want:

Create a microservices e-commerce architecture with API Gateway, auth/user/order/product/payment services,
Kafka message queue, notification service, and separate databases for each service

The skill plans the layout, generates the .drawio XML, exports to your chosen format, self-checks, and lets you iterate.
